Viewing a Previous Run

This is what a typical run entry looks like. Details that a user previously entered using the add run function are queried from the database and displayed for the user's review. My favorite parts of viewing a previous run are the route from mapmyrun which had been previously stored (see the routes tab), the instagram photo, and the location/terrain/and route badges that display (they are unique for each running condition! Go for a run in a new place or on a new terrain and see the different badges!) I also like the svg bubbles that indicate how the user said that they were feeling. (Red indicates a low score, Green indicates a high score.)
